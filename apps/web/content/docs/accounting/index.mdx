# Accounting overview

BookOne has **one shared ledger** (`transactions`, `journal_entries`, `journal_lines`).

## Two ways in

| Highway | Best for |
|---------|----------|
| **Commercial modules** | Invoices, bills, stock, AR/AP documents |
| **Simple Entry** (home) | Ad-hoc expense, transfer, owner contribution, quick sale |

**Rule:** do not post the same economic event twice (e.g. Simple Entry sale + sales invoice). Simple Entry warns on possible duplicates.

## Core screens

- Journal, Transactions, Accounts, Reports, Reconciliation
