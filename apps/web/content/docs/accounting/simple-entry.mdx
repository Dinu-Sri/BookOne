# Simple Entry

Home screen modes:

| Mode | Typical journal |
|------|-----------------|
| Money in → New sale | Dr cash / Cr 4000 |
| Money in → Customer payment | Dr cash / Cr 1300 (no invoice allocation) |
| Money in → Owner contribution | Dr cash / Cr 3000 |
| Money out | Dr expense / Cr cash |
| Move money | Dr to / Cr from |
| Invoice/Bill → Customer | Dr 1300 / Cr 4000 |
| Invoice/Bill → Vendor bill | Dr expense / Cr 2100 |

## When to use Sales/Purchase instead

- Stocked goods  
- Multi-line bills  
- VAT tax invoices  
- Matching payments to documents  

## Force duplicate

If Simple Entry matches an existing commercial document same day/party/amount, you must check **force** to post anyway (not recommended).
