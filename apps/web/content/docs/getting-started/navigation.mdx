# Navigation tips

## Sidebar suites

The left shell groups modules (Sales, Purchase, Inventory, Company, and others your company has enabled).  
Click a suite, then a screen (e.g. **Sales → Invoices**).

## Lists vs forms

Most modules follow the same pattern:

1. **List** — search, sort, open a row  
2. **New** — create form with lines  
3. **Detail** — view, print, convert, pay  

## Print

Invoices and purchase docs often have a **Print** route under the document (e.g. `/sales/invoices/[id]/print`).

## Docs shortcut

Bookmark **`/docs`** for this documentation. You can keep it open beside BookOne while learning a process.
