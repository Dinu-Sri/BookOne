# Master wipe

On the Health Check page **Danger zone**:

1. Company must be **Staging**.  
2. Click **Master wipe all data…**  
3. Confirm, then type **`MASTER RESET`**.  

## Deleted

Ledgers, documents, products, stock, parties, payments, POS shifts, health-check history, uploads (when storage configured).

## Kept

Company profile, tax, brands, locations, domains, FYs, chart of accounts, users, module settings, POS registers.

**Never** run master wipe on a real production company.
