# BookOne documentation

Welcome. This site explains **how to use BookOne** day to day — for company users, accountants, and cashiers.

## What is BookOne?

BookOne is an accounting & ERP system for your business. You can:

- Record **sales** (quotes → orders → invoices → payments)
- Run **purchases** (PO → GRN → bills → pay vendors)
- Track **inventory** (products, stock, transfers, adjustments)
- Post through a **double-entry ledger** (journals, trial balance, reports)
- Use **POS** for counter sales
