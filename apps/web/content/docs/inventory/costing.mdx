# Costing methods

**Company → Inventory settings → Costing method**

| Method | On purchase / GRN stock-in |
|--------|----------------------------|
| **Last** | Product `unitCost` becomes this receipt’s cost |
| **Average** | Weighted average of qty on hand before receipt + new receipt |

COGS on sales uses the product’s current `unitCost` at sale time.
