# Inventory overview

## Product types

| Type | Stock qty | COGS on sale |
|------|-----------|--------------|
| Physical | Yes | Yes |
| Digital | No | No |
| Service | No | No |

## Screens

- **Products** — master data, accounts, opening qty  
- **Stock levels** — qty by location  
- **Stock ledger** — movements  
- **Transfers** — location A → B (no GL)  
- **Adjustments** — qty ± with GL vs inventory
