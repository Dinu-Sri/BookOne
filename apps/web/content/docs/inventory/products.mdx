# Products

## Create a product

1. **Inventory → Products → New**.  
2. SKU, name, type (physical/digital/service).  
3. Unit cost, sell price.  
4. Account codes (revenue, COGS, inventory) — defaults 4000 / 5000 / 5100.  
5. Optional opening quantity (physical) — posts opening stock GL when qty & cost > 0.  
6. Save.

## Free-text lines on documents

On sales/purchase forms you can type a free-text line and **save as product** (quick create) so stocked items stay on the master list.
