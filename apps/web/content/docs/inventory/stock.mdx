# Stock levels and ledger

## Levels

**Inventory → Stock levels**

- Qty on hand by product and location  
- Stock value ≈ qty × unit cost  
- Low stock filter when reorder level is set  

## Ledger

**Inventory → Stock ledger**

- Sales, purchases, GRNs, transfers, adjustments  

## Negative stock

If **Inventory settings → Negative stock = Block**, sales/returns/adjustments that would go below zero are rejected.
