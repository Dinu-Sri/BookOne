# Stock transfers

1. **Inventory → Transfers → New**.  
2. Choose **from** and **to** locations.  
3. Add physical products and quantities.  
4. Save.

## Rules

- Total company qty **unchanged**  
- **No journal** (balance sheet inventory total same)  
- Source location must have enough qty if negative stock is blocked
